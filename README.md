[德州仪器TI](http://www.ti.com.cn/)是全球领先的数字信号处理与模拟技术半导体供应商，也是世界十大半导体公司排名前三的存在。是模拟和数字嵌入式及应用处理半导体解决方案的领导者。

TI的核心业务可以分为：模拟产品（Analog）、嵌入式处理器（Embedded Processors）和其他（DLP ＆ EdTech）。
